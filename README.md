## 職務経歴

### 株式会社 GAGA(2025/03〜2025/12)

#### 担当案件： 大手自動車向け物流自動化案件(2025/04〜2025/12)

### 業務内容

AMR（自律搬送ロボット）とWCS（Warehouse Control System）を組み合わせた物流自動化システムの運用・保守業務に従事。

システム監視、障害対応、テスト仕様書作成、単体テスト / 結合テストを担当しました。

また、シミュレーション環境の立ち上げ手順書や障害発生時の復旧手順書作成にも携わり、現場チームと連携しながら安定稼働維持に貢献しました。

### 担当業務

- システム監視
- 障害対応
- テスト仕様書作成
- 単体テスト / 結合テスト

### 背景・課題

搬送ロボットによる部品供給において、本来時間内に届くべき部品が遅延する問題が発生していました。

大量生産・本番運用の環境では、部品供給の遅延が生産効率に影響するため、安定して時間内に供給できる状態が求められていました。

### 施策

問題発生時には、監視画面や搬送状況を確認し、どの工程で滞留や遅延が発生しているのかを切り分けました。

そのうえで、関係者へのエスカレーションや運用フローの見直しを提案し、部品供給が安定して時間内に間に合う状態への改善を行いました。

### 成果

結果として、部品供給にかかる時間短縮につながり、本番運用における安定稼働の維持に貢献しました。

### 有限会社ル・スリジェダムール洋菓子店(2020/03〜2024/05)

### 担当業務

- 製品の生産業務
- 生産管理（週間生産表の作成・運用）
- 在庫管理 / 発注業務
- 予約管理・受注対応

### 背景・課題

7店舗を展開する企業のセントラル工房にて、各店舗からのオーダーに応じた製品・半製品の生産を担当していました。

所属部署では経験の浅いメンバーが多く、オーダー内容の把握が難しい状況だったため、生産効率や情報共有に課題がありました。

### 施策

課題解決のため、必要な製品や数量を可視化する週間生産表を作成しました。

生産に必要な情報を整理・共有することで、メンバー全体がオーダー内容を把握しやすい状態への改善を行いました。

### 成果

結果として、部署内での情報共有が円滑になり、業務効率向上やチーム内コミュニケーションの改善につながりました。
